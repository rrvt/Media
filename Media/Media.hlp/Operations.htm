Media Operations

Find Media

Search for a media to see if it is the database.

Add Media Description

Add a new media description to the database.

Edit Media Description

Edit an existing media description.

Delete Media Description

Delete an existing media description.

Refresh Display

Refresh the display in the window.

Save File

Displays a Save File Dialog and saves the data to the database file.

Print

Will do all the normal things to print the text in the window.

Options

The options dialog allows specifying the Display and Printer Font Size Scale Factor, the printer orientation and the printer margins.

About

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit

Save the data to the current database and exit the database.
